# Melvin-Pro-Supreme-v4.2
Sesuai namanya, tool sederhana ini saya buat untuk membantu teman-teman creator di Roblox yang sering kesulitan saat ingin upload audio/musik tapi terus-menerus kena sensor otomatis (copyright). Tool ini bekerja dengan mengubah "sidik jari digital" (audio fingerprint) file musik kamu agar sulit dideteksi oleh bot scanner Roblox.
