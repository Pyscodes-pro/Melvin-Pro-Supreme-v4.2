## 📌 Overview

MELVIN PRO SUPREME AUDIO TOOLKIT adalah aplikasi desktop berbasis Python yang dirancang untuk membantu kreator mengoptimalkan file audio melalui berbagai fitur pemrosesan suara modern.

Tool ini mendukung workflow audio untuk kebutuhan:

* Optimasi kualitas suara
* Remastering sederhana
* Penyesuaian karakter audio
* Persiapan audio untuk berbagai platform
* Eksperimen sound design

> Gunakan hanya pada audio yang Anda miliki, buat sendiri, atau telah memperoleh izin penggunaannya.

---

## ✨ Features

* 🎛️ **Dual-Column Interface** — Tata letak horizontal yang efisien dan mudah digunakan
* 🎵 **Pitch Adjustment** — Mengubah karakter suara tanpa merusak kualitas vokal
* ⏱️ **Time Stretching** — Menyesuaikan tempo audio secara fleksibel
* 🔊 **Bass Enhancement** — Menambahkan kedalaman frekuensi rendah
* 🎧 **Stereo Widening** — Memperluas ruang stereo untuk pengalaman mendengarkan yang lebih imersif
* ⚡ **Real-Time Progress Tracking** — Pantau proses rendering secara langsung
* 🚀 **Multi-Threaded Processing Engine** — Performa lebih cepat dan responsif
* 📁 **Large File Support** — Mendukung pemrosesan audio berdurasi panjang
* 🎚️ **Custom Audio Effects** — Konfigurasi parameter audio sesuai kebutuhan

## 📦 Installation

### 1. Install Python

Unduh dan instal Python versi terbaru:

* https://www.python.org/downloads/

Pastikan opsi **Add Python to PATH** dicentang saat instalasi.

### 2. Install Dependencies

Buka terminal atau Command Prompt di folder proyek, lalu jalankan:

```bash
pip install pydub pedalboard audioop-lts
```

### 3. Setup FFmpeg (Required)

1. Unduh FFmpeg dari situs resminya.
2. Ekstrak file ZIP.
3. Buka folder `bin`.
4. Salin file berikut:

```text
ffmpeg.exe
ffprobe.exe
```

5. Tempelkan kedua file tersebut ke folder yang sama dengan script utama.

Struktur folder yang disarankan:

```text
project/
├── RobloxAudioTool.py
├── ffmpeg.exe
├── ffprobe.exe
└── README.md
```

## 🚀 Usage

Jalankan aplikasi menggunakan perintah:

```bash
python RobloxAudioTool.py
```

Langkah penggunaan:

1. Buka aplikasi.
2. Pilih file audio (`MP3`, `WAV`, atau `OGG`).
3. Sesuaikan parameter audio.
4. Klik **START PROCESS**.
5. Tunggu hingga proses selesai.
6. File hasil akan disimpan secara otomatis.

## ⚙️ Recommended Settings

### Balanced Preset

* Pitch: `0.95`
* Speed: `1.02`
* Reverb: `10%`
* Stereo Width: `Medium`

### Enhanced Preset

* Pitch: `1.05`
* Speed: `1.08`
* Bass Boost: `Moderate`
* Stereo Width: `High`

> Sesuaikan pengaturan berdasarkan karakteristik setiap audio.
